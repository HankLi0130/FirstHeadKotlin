# Kotlin 學習分享

Kotlin 是一個很進步的程式語言，除了完全支援 Java 外，還提供了現代程式設計的觀念，例如 Lambda, Streaming, Coroutines ...，加上 Google 在 Google I/O 2017 上宣布 Kotlin 為 Android 的主要開發語言，更成為開發行動裝置的利器，現在除了積極發展多平台開發 (iOS, Android)之外，也在嵌入式系統方面持續發展，只能說 Kotlin 的前途大好，有興趣的同學，快上車吧～
